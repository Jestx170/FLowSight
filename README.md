## ส่วนที่ 1 — เนื้อหาสำหรับเว็บไซต์ (Website Content)

### FlowSight คืออะไร?

**FlowSight** คือแพลตฟอร์มวิเคราะห์พฤติกรรมลูกค้าด้วยปัญญาประดิษฐ์ (AI) แบบเรียลไทม์ ออกแบบมาเพื่อช่วยให้ธุรกิจค้าปลีก ร้านอาหาร และสถานประกอบการต่าง ๆ เข้าใจพฤติกรรมของผู้ใช้บริการในพื้นที่ได้อย่างลึกซึ้ง โดยไม่ต้องพึ่งพาการสังเกตของมนุษย์ตลอดเวลา

---

### ฟีเจอร์หลัก

#### 🧠 Dashboard อัจฉริยะพร้อม Activity Log
แดชบอร์ดแบบ Real-time ที่แสดงพฤติกรรมของลูกค้าและพนักงานในพื้นที่ครอบคลุมทุกโซน ระบบตรวจจับและบันทึกพฤติกรรมอัตโนมัติ เช่น:
- **การรอคิว (Waiting)** — ตรวจจับลูกค้าที่รอบริเวณทางเข้าหรือจุดรอ พร้อมแจ้งเตือนเมื่อรอนานเกินกำหนด
- **การอยู่นิ่งนาน (Loitering)** — ระบุบุคคลที่หยุดอยู่ในพื้นที่นานผิดปกติ
- **การใช้งานโซนต่าง ๆ (Zone Activity)** — วิเคราะห์ความหนาแน่นและรูปแบบการเคลื่อนที่ในแต่ละโซน เช่น ทางเข้า, โซนนั่ง, โซนพนักงาน
- **การแจ้งเตือนอัตโนมัติ (Smart Alerts)** — ส่งสัญญาณเตือนทันทีเมื่อมีเหตุการณ์ที่ต้องการการดูแล เช่น ลูกค้าต้องการความช่วยเหลือ หรือนั่งนานเกินเวลาที่กำหนด

#### 🔒 Face Anonymization — ปกป้องความเป็นส่วนตัวอย่างเต็มรูปแบบ
FlowSight มีระบบ **เบลอใบหน้าอัตโนมัติ (Face Anonymization)** ที่ทำงานบนฟีดกล้องโดยตรง โดยจะประมวลผลและปิดบังใบหน้าของทุกบุคคลก่อนที่ภาพจะแสดงผลหรือถูกบันทึก ช่วยให้ธุรกิจสามารถวิเคราะห์พฤติกรรมได้อย่างมีประสิทธิภาพ โดยยังคงปฏิบัติตามกฎหมายคุ้มครองข้อมูลส่วนบุคคล (PDPA) และมาตรฐานความเป็นส่วนตัวระดับสากลได้อย่างเคร่งครัด

#### 📊 กราฟและการวิเคราะห์รายชั่วโมง (Detailed Graphs & Hourly Tracking)
- **กราฟการเข้าใช้งานรายชั่วโมง** — ดูแนวโน้มการเข้า-ออกของลูกค้าตลอดทั้งวัน
- **Peak Hour Detection** — ระบุช่วงเวลาที่มีลูกค้าสูงสุดโดยอัตโนมัติ
- **สรุปพฤติกรรมรายวัน** — รายงานครบถ้วนเกี่ยวกับจำนวนลูกค้า, พฤติกรรมเด่น, โซนที่ได้รับความนิยม และอัตราการแปลงยอดขาย
- **AI Insight** — ระบบวิเคราะห์ข้อมูลด้วย AI สรุปผลและแนะนำแนวทางปรับปรุงการดำเนินงานโดยอัตโนมัติ

#### ⚙️ ติดตั้งง่าย ใช้งานได้ทันที
FlowSight รองรับการใช้งานบนระบบ Windows พร้อม Installer สำเร็จรูป ไม่ต้องการความรู้ด้านโปรแกรมมิ่ง รองรับกล้อง IP หลายตัวพร้อมกัน และสามารถปรับแต่งโซนและพฤติกรรมที่ต้องการติดตามได้ตามความต้องการของแต่ละธุรกิจ

### 💼 ธุรกิจของคุณกำลังสูญเสียรายได้โดยไม่รู้ตัว

ทุกวันที่ผ่านไป ลูกค้าเดินเข้าร้านของคุณ บางคนรอนาน บางคนต้องการความช่วยเหลือแต่ไม่มีพนักงานเข้าถึง บางคนออกไปโดยที่คุณไม่รู้ว่าเกิดอะไรขึ้น

**FlowSight** คือสายตาที่สามที่คุณต้องการ ระบบที่ไม่เคยหลับ ไม่เคยพลาด และพร้อมรายงานให้คุณทราบทุกสิ่งที่เกิดขึ้นในพื้นที่ธุรกิจของคุณแบบ Real-time

---

### ✅ ทำไมธุรกิจชั้นนำเลือก FlowSight?

**เพราะการวัดผลคือจุดเริ่มต้นของการเติบโต**

FlowSight ไม่ได้เป็นแค่กล้องวงจรปิด แต่คือระบบวิเคราะห์พฤติกรรมที่เปลี่ยนข้อมูลจากกล้องธรรมดาให้กลายเป็น **Intelligence** ที่คุณนำไปตัดสินใจทางธุรกิจได้จริง

- ✔ รู้ว่าลูกค้ารอนานแค่ไหน — ปรับจำนวนพนักงานได้อย่างแม่นยำ
- ✔ รู้ว่าโซนไหนคึกคักที่สุด — วางสินค้าและจัดพื้นที่ได้อย่างมีกลยุทธ์
- ✔ รู้ว่าลูกค้าต้องการความช่วยเหลือแบบเรียลไทม์ — ลดโอกาสสูญเสียยอดขาย
- ✔ รู้ข้อมูลเชิงลึกรายชั่วโมง — วางแผนโปรโมชั่นและ staffing ได้ตรงจุด

### 🔐 ความเป็นส่วนตัว ไม่ใช่ทางเลือก — แต่คือมาตรฐาน

ในยุคที่ความเป็นส่วนตัวของข้อมูลเป็นเรื่องสำคัญ FlowSight ออกแบบมาพร้อมระบบ **Face Anonymization** ตั้งแต่ต้น ระบบจะเบลอใบหน้าทุกคนโดยอัตโนมัติ ทำให้คุณสามารถวิเคราะห์พฤติกรรมได้เต็มรูปแบบโดย **ไม่เก็บข้อมูลชีวมิติ (Biometric Data)** ของลูกค้าเลยแม้แต่น้อย มั่นใจได้ว่าธุรกิจของคุณปลอดภัยจากความเสี่ยงด้านกฎหมาย PDPA ทั้งในปัจจุบันและอนาคต
